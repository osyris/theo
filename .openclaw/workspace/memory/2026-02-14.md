### Технические находки
- Fragrantica фото: `https://fimgs.net/mdimg/perfume/375x500.{ID}.jpg` — работает без блокировки
- Fragrantica HTML: заблокирован Cloudflare (403 для всех URL, включая несуществующие)
- Parfumo.com: работает для нот! Парсинг: `re.split(r'(Top Note|Heart Note|Base Note)', html)` + `re.findall(r'alt="([^"]+?)"\s*loading="lazy"\s*class="np', part)`
- Parfumo тоже блокирует ~60% запросов (Cloudflare), но ~40% проходят
- Для поиска Fragrantica ID: Yahoo search `site:fragrantica.com "perfume name"` — иногда работает
